Exchange Rate API

4 para birimi arasında EUR, USD, TRY, GBP dönüşüm için güncel exchange değerlerini gönderir.

(Diğer üç para biriminden hangi para birimine dönüştürmek istiyorsanız parametre olarak onu girin.


Buradaki herhangi bir API isteği www.currencyconverterapi.com sitesine ayrı ayrı üç istek göndermektedir. ww.currencyconverterapi.com sitesi ise toplamda bir saatte en fazla 100 istek kabul etmektedir. O yüzden, github'dan projeyi indirip kendi API key'inizi kullanmanızı istirham ederim.
